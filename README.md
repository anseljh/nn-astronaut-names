# nn-astronaut-names

I used a list of US astronaut names and used it to train a neural network
to generate more astronaut names.

Some good results include:

* James Pullin J. Nimhfard
* Harges Brinen, Jr.
* Tart Manraugty D. GritemeLy
* Risk Reighurass
* Oovis Scactall
